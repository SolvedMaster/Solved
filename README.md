```java
public class Main{
    public static void main(String[] args) {
        /*

    }
```

- #### 클래스 명명 규칙:    
  - 숫자로 시작하면 안된다.
  - 첫문자는 대문자로 한다.
  - Q문제번호_푼사람 ex) Q1000_thdqudgns
