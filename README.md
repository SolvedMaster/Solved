# Solved

- #### 백준 사용방법: 
  
```java
public class Main{
    public static void main(String[] args) {
        /*
         * 코드 작성
         */
    }
}
```

- #### 클래스 명명 규칙:    
  - 숫자로 시작하면 안된다.
  - 첫문자는 대문자로 한다.
  - Q문제번호_푼사람 ex) Q1000_thdqudgns
  - 문제번호가 앞에 있어야 문제별 분류가 된다.
